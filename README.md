# Naxa Assesment

## Installation

To get started with Naxa Services, follow these steps:

1. Clone the repository:

    ```
    git clone https://github.com/HariomGupta123/NaxaServices.git

    cd NaxaServices/assessmentTask
 
    npm install
    ```
    ```
    npm start
    ```
    or
    ```
    npm run dev
    ```
